# gg-hub — Платформа для игровых гильдий

## Назначение

**gg-hub** — веб-платформа (хаб) для поиска, создания и управления игровыми гильдиями в ММОРПГ. Объединяет игроков и гильдии из разных игр в единое пространство с инструментами организации, коммуникации и контента.

### Какие проблемы решает

- **Фрагментация сообществ** — игрокам сложно найти гильдию под свой стиль игры, расписание, язык и ценности. Встроенные инструменты поиска в играх примитивны.
- **Отсутствие инструментов управления** — лидерам и офицерам не хватает средств для организации вне игры (Discord + Google Таблицы — не всегда удобно).
- **Видимость и репутация** — новым и маленьким гильдиям сложно заявить о себе; игрокам трудно оценить атмосферу гильдии до вступления.
- **Кросс-игровое присутствие** — игрок часто играет в несколько игр, но его репутация и связи не сохраняются при переходе.
- **Поиск команды** — не всегда нужна гильдия, иногда нужно найти группу на конкретный рейд или активность.
- **Удержание интереса вне игры** — в перерывах между играми сообщество распадается без площадки для общения.

### Для кого

- Игроки ММОРПГ, ищущие гильдию или команду
- Лидеры и офицеры гильдий, которым нужны инструменты управления
- Кросс-игровые сообщества, существующие в нескольких играх одновременно

### Поддерживаемые игры

- Black Desert
- World of Warcraft
- Guild Wars 2
- Lineage 2
- Aion 2

Список игр расширяется через админку.

## Реализованный функционал

### Инфраструктура

- **Docker-окружение** — полная контейнеризация: nginx, PHP-FPM + очередь, MariaDB, Redis, фронтенд (Node/Vite), phpMyAdmin, socket-сервер, Certbot (SSL)
- **Деплой на продакшен** — сайт доступен на домене `gg-hub.ru`

### Пользователи и авторизация

- **Регистрация и авторизация** — через Laravel Fortify
- **Смена пароля**
- **Профиль пользователя** — аватар, информация, привязанные персонажи
- **Учёт часового пояса** — для корректного отображения времени событий
- **Управление пользователями в админке** — список пользователей, страница информации о пользователе
- **Система ролей и прав** — роли с набором прав как для администрации сайта, так и для участников гильдии в рамках конкретной гильдии

### Игры и персонажи

- **Управление играми в админке** — добавление игр с изображением, классами, локализациями, серверами
- **Классы персонажей** — изображение, название, цвет; возможность группировки классов; настройка количества классов на персонажа
- **Серверы** — привязка серверов к играм и локализациям
- **Персонажи пользователей** — создание персонажа с привязкой к игре, локализации, серверу, классу; изображение, имя, теги
- **Фильтрация персонажей** — по игре, серверу, классу, имени; по умолчанию выбрана текущая игра
- **Теги** — управляются администратором, имеют изображение, настраиваемое ограничение количества

### Гильдии

- **Список гильдий** — отображение гильдий текущей игры; на общей странице — группировка по названию с указанием игр
- **Карточка гильдии** — полная информация о гильдии
- **Создание/редактирование/удаление гильдии** — изображение, игра, локализация, название, описание, теги, статус
- **Навигация по гильдиям** — если пользователь в одной гильдии — одно подменю; если в нескольких — перечисление гильдий с подменю; иконка короны для лидера
- **Правило принадлежности** — один аккаунт может состоять в нескольких гильдиях, но конкретный персонаж — только в одной (в рамках игры, локализации, сервера)

### Заявки и приглашения

- **Карточка заявки в гильдию** — описание, список принимаемых классов
- **Кастомные поля заявки** — администратор гильдии настраивает поля для заполнения (типы: input, select, image, multiselect)
- **Подача заявок** — пользователь заполняет заявку и отправляет
- **Приглашения** — гильдия может отправлять приглашения игрокам
- **Комментарии к заявке** — обсуждение заявки участниками гильдии

### События и рейды

- **Календарь эвентов** — разовые, ежедневные, еженедельные, ежемесячные события
- **Создание эвентов** — участники, скриншоты посещаемости, описание
- **Рейды** — формирование рейда с добавлением участников гильдии

### Контент и модерация

- **Блог** — посты трёх уровней видимости: гильдейские (внутри гильдии), игровые (в рамках игры), общие (весь сайт)
- **Лента пользователя** — единая лента всех постов с фильтрацией по типу
- **Администрирование комментариев** — модерация с указанием причины удаления
- **Голосование (опросы)** — создание опросов в рамках гильдии

### Оповещения

- **Система оповещений** — уведомления пользователей о событиях на платформе
- **Telegram-интеграция** — опциональная отправка уведомлений в Telegram

### Администрирование

- **Разделение админки и пользовательской части**
- **Управление играми, серверами, классами, пользователями, постами, комментариями**

## Планируемый функционал

### Личный кабинет пользователя
Расширенный личный кабинет с историей персонажей, активности и настройками.

### Оповещения (редизайн)
Переработка системы оповещений на компонент Sonner (shadcn) для улучшения UX.

### Интеграция рейдов и аукциона
Выгрузка участников из истории рейда и загрузка в аукцион.

### Лайки/дизлайки к заявкам
Возможность участникам гильдии ставить лайк или дизлайк к заявке на вступление.

### Рулетка
Интерактивный барабан в реальном времени через Socket.IO — вращение видно всем на странице, но запускать может только один человек.

### Верификация владельца гильдии
Галочка подтверждения реального владельца гильдии.

### Чат с комнатами
Текстовый чат гильдии с системой комнат.

### Голосовой чат (WebRTC)
Голосовой чат с комнатами на базе WebRTC.

### Фильтр гильдий (рефакторинг)
Вынесение фильтра гильдий в отдельный переиспользуемый компонент.

## Позиционирование и конкурентные преимущества

### Аналоги на рынке

| Платформа | Особенности |
|-----------|-------------|
| Guilded.gg | Полноценная платформа для сообществ (аналог Discord для геймеров), сложная для небольших гильдий |
| Raider.io / WoWProgress | Узкоспециализированные агрегаторы прогресса для WoW |
| Guild.space | «Linktree для гильдий» — лендинги без глубокого функционала |
| MMORPG.com | Портал с форумами, но устаревший UX |
| Discord | Главный конкурент и одновременно точка интеграции |

### Преимущества gg-hub

- **Кросс-игровой профиль** — единый аккаунт игрока с персонажами в разных играх и историей переходов между гильдиями
- **Кросс-игровые гильдии** — сообщества, существующие в нескольких играх одновременно
- **Современный UI/UX** — в отличие от устаревших каталогов 2010-х годов
- **Фокус на русскоязычную аудиторию** — ниша, освободившаяся после потери актуальности MMORPG.ru
- **Структурированное управление** — не просто чат, а инструменты для рекрутинга, рейдов, календаря и модерации
- **Дополнение к Discord** — не замена, а «витрина» и организационный хаб, который ведёт людей в Discord гильдии
