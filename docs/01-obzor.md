# 1. Обзор продукта

## Назначение

**gg-hub** — веб-платформа (хаб) для поиска, создания и управления игровыми гильдиями в ММОРПГ. Объединяет игроков и гильдии из разных игр в единое пространство с инструментами организации, коммуникации и контента.

## Какие проблемы решает

| Проблема | Решение на платформе |
|----------|----------------------|
| Фрагментация сообществ | Каталог гильдий с фильтрами по игре, серверу, тегам |
| Нет инструментов управления вне игры | Календарь, рейды, заявки, роли, хранилище, ДКП |
| Низкая видимость новых гильдий | Публичные карточки, журнал, общие посты |
| Кросс-игровое присутствие | Один аккаунт, персонажи в разных играх |
| Поиск команды на активность | Заявки, приглашения, календарь событий |
| Распад сообщества между играми | Лента, опросы, посты гильдии |

## Целевая аудитория

- Игроки ММОРПГ, ищущие гильдию или команду
- Лидеры и офицеры гильдий
- Кросс-игровые сообщества

## Реализованный функционал

### Пользователи и авторизация

- Регистрация и вход (Laravel Fortify, cookie-сессия)
- OAuth: Yandex, VKontakte
- Верификация email
- Смена пароля, профиль (аватар, часовой пояс)
- Система ролей сайта (админка на субдомене `admin.*`)

### Игры и персонажи

- Админка: игры, локализации, серверы, классы персонажей
- Персонажи пользователя: игра, локализация, сервер, класс, теги, изображение
- Правило: **один персонаж — одна гильдия** в контексте игры/локализации/сервера; **один аккаунт** может состоять в нескольких гильдиях через разных персонажей

### Гильдии

- Список и карточка гильдии, создание/редактирование
- Состав (roster), роли и права внутри гильдии
- Заявки и приглашения с кастомными полями формы
- Комментарии к заявкам, голосование по заявкам
- Теги состава (гильдейские / общие / личные)
- Настройки гильдии (в т.ч. Discord webhook, флаг ДКП)

### События и рейды

- **Календарь** — разовые, ежедневные, еженедельные, ежемесячные события; участники; уведомления в Discord
- **История событий** — фиксация посещений, скриншоты, начисление ДКП
- **Рейды** — дерево рейдов, состав по слотам, синхронизация в realtime

### Хранилище и ДКП

- Каталог предметов, тиры, выдачи, журнал ДКП
- Подробно: [guild-bank-and-dkp.md](guild-bank-and-dkp.md)

### Контент

- Посты трёх уровней: гильдия / игра / глобальный сайт
- Модерация, комментарии, просмотры
- Опросы (polls) в гильдии

### Оповещения

- In-app уведомления + push через Socket.IO
- Telegram и Discord через **notification-gg-hub**

### Администрирование

- Субдомен `admin.*`: игры, пользователи, роли, права, посты, комментарии, опросы, теги, статистика CTA лендинга

## Планируемый функционал

- Расширенный личный кабинет
- Редизайн toast-уведомлений (Sonner)
- Интеграция рейдов и рулетки (выгрузка участников)
- Лайки/дизлайки к заявкам
- Рулетка (частично есть UI + socket)
- Верификация владельца гильдии
- Чат с комнатами, голосовой чат (WebRTC)
- Рефакторинг фильтра гильдий

## Конкуренты и позиционирование

| Платформа | Особенности |
|-----------|-------------|
| Guilded.gg | Мощная, тяжёлая для малых гильдий |
| Raider.io / WoWProgress | Узкая специализация (WoW) |
| Guild.space | Лендинги без глубокого функционала |
| Discord | Конкурент и точка интеграции |

**Преимущества gg-hub:** кросс-игровой профиль, современный UI, фокус на RU-аудиторию, структурированное управление, дополнение к Discord (не замена).
